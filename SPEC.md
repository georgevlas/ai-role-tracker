# Product Spec: AI Role Tracker

## Problem
I need a structured way to manage job opportunities, recruiter conversations, company research, follow-ups, and application status.

## Target User
George, a technology executive managing multiple CTO/CIO/technology leadership opportunities.

## Core Use Cases
- Add a new role opportunity
- Track recruiter and company details
- Record notes and next actions
- Set follow-up dates
- Track status from lead to closed
- Search and filter opportunities

## Core Features
- Role/opportunity list
- Add/edit opportunity
- Status field
- Contact/recruiter details
- Notes field
- Follow-up date
- Search/filter

## Out of Scope for Version 1
- AI summarisation
- Email integration
- Calendar integration
- Mobile app
- Multi-user access

## Success Criteria
- I can enter and update all active roles
- I can see next follow-ups
- I can filter by status
- I can avoid losing track of opportunities
